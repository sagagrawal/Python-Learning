Black Jack Game has been the project of my learning and developing Python skills, getting used to code with python semantic ans structure.

This a Casino game if you are familiar with this. Anyone using the project to play would definitely have fun.

Steps to Run this Project:
1. Download this repo to your local system having Python installed(If not already, you will have to install Python).
2. Unzip the Downloaded folder
3. Go to Command Prompt -> Navigate to the Directory containing the files of this project
4. Type in Command: "python.exe Run.py"
5. Enjoy the game

P.S: You can also refer to OutputSample.txt file that contains a Sample Run output of this Game.
